phpbf
=====

PHPBF is a BrainFuck compiler, written in PHP, that compiles BrainFuck to C.

Note: This version has only been tested with PHP version 5.5.1

Examples of Use
---------------
```
phpbf examples/test.bf test.c
clang test.c -o test
./test
```
